hs-login-membership
===================

HealthSPORT Membership Plugin. A plugin for <a url="http://healthsport.com">HealthSPORT.com</a> that allows members to login to the GroupX member system (via an
URL embedded in an iframe) and create an account via a google spreadsheet also embedded in an iframe. Both embedded values are settable via the Settings menu
in the WordPress Dashboard.
